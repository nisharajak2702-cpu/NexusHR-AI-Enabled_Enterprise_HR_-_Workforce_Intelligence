NexusHR Project Handover Document

Project Overview

NexusHR is a Spring Boot + PostgreSQL + JWT based Human Resource
Management System (HRMS) with a React (Vite) frontend.

Backend

-   Spring Boot 3.5.x
-   Java 21
-   PostgreSQL
-   Spring Security + JWT
-   JPA/Hibernate
-   Swagger
-   OpenPDF (Payslips)
-   Excel Export
-   Audit Logging

Frontend

-   React (Vite)
-   React Router
-   Axios
-   Material UI
-   JWT Authentication

Important Notes

-   Backend login works and returns token, username, role, message.
-   React runs on http://localhost:5173
-   Backend runs on http://localhost:8080
-   JWT must be sent as Authorization: Bearer .
-   CORS must be enabled for localhost:5173.
